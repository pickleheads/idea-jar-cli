# Pickle Jar CLI

> A place to store our :shit:ty ideas

## Installation

Install with NPM:

```bash
npm install -g pickle-jar-cli
```

## Usage

```
pickle-jar <cmd>

  Commands:
    add             Add a new idea
    ls              List existing ideas
    repl            Enter REPL mode

  Options:
    --help, -h      Show help
    --version, -v   Show version

```
